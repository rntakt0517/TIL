## message 수정하기

1. 명령문

```bash
git commit --amend
```

2. message 수정

3. `:wq` 작성

> 이름만 바뀌는 게 아니라 새로운 commit id 가짐

---

## commit 전체 수정

1. 새로운 수정본 Staging Area에 옮기기
2. 명령문

```bash
git commit --amend
```

3. `:wq`작성

> 실수한 commit에 **추가로 올리는 것이 아니라** 한 commit으로 만들어 올리기
