## 반복과 재귀

- 반복 (Iteration)
  
  - 루프 (for, while 구조)
  
  - 반복문은 코드를 n번 반복시킬 수 있음

- 재귀 (Recursion)
  
  - 하나의 큰 문제를 해결할 수 있는 더 작은 문제로 쪼개고 결과들을 결합
  
  - n중 반복문을 만드러낼 수 있음
  
  > int 타입 객체를 함수에 전달하면 **값**만 복사
  > 
  > 함수가 끝나면 해당 함수를 호출했던 곳으로 돌아옴
  
  - 재귀호출 공부의 시작 : 무한 재귀호출을 막는 것! **<mark>기저조건</mark>** (base case)

## 순열

- 순열이란?
  
  - 서로 다른 N개에서 R개를 중복없이, 순서를 고려하여 나열하는 것
  
  - 중복 순열 코드에서 중복을 제거하는 코드 추가
  
  > 전역 리스트를 사용하면 이미 선택했던 숫자인지 구분 가능

- 중복순열이란?
  
  - 서로 다른 N개에서 R개를 중복을 허용하고, 순서를 고려하여 나열하는 것

- 구현 원리
  
  1. 재귀호출을 할 때마다, 이동 경로를 흔적으로 남김
  
  2. 가장 마지막 레벨에 도착했을 때, 이동 경로 출력
