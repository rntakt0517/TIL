## 부분 집합

집합에 포함된 원소들을 선택하는 것 (아무것도 선택하지 않은 경우도 집합에 포함 : 공집합)

- 구현 방법
  
  1. 완전 탐색
     
     - 재귀호출을 이용한 완전탐색
     
     - 실전 보다는 완전 탐색 학습용으로 추천하는 방법
  
  2. **<mark>Binary Counting</mark>**
     
     - 2진수 & 비트 연산을 이용
     
     - 부분 집합이 필요할 때 사용하는 추천 방법
     
     - Branch(후보군), Level(원소 개수)

## 조합

 서로 다른 n개의 원소 중 r개를 **순서 없이** 골라낸 것

- 구현 방법
  
  1. 뽑는 호보군 개수 만큼 n중 for문
  
  2. 재귀호출 구현이 필요
     
     ```python
     arr = ['A', 'B', 'C', 'D', 'E']
     path = []
     n = 3
     
     def run(lev, start):
          if lev == n:
              print(*path)
              return
     
          for i in range(start, 5):
              path.append(arr[i])
              run(lev + 1, i + 1)
              path.pop()
     
     run(0, 0)
     ```

## 탐욕 알고리즘

Greedy (욕심장이기법)

결정이 필요할 때, 현재 기준으로 가장 좋아보이는 선택지로 결정하여 답을 도출

- 거스름돈 줄 때 완전 탐색이 아니라 큰 돈부터 계산하는 것처럼..

- 그러나 10원 50원 70원이 있을 때 예외가 발생

> 그리디가 성립하는 경우 : 모든 동전이 배수 관계인 경우
> 
> 성립하지 않는 경우 : 모든 동전이 배수 관계가 아닌 경우

- 문제 예시
  
  - 화장실 문제 : sort하기
  
  - Fractional Knapsack 문제 : Kg 당 가격이 가장 높은 물건을 최대한 담기
  
  - 회의실 배정 문제 : 종료시간이 가장 빠른 회의를 먼저 선택
