## Python을 배우는 이유

- 쉽고 간결한 직관적인 문법

- 파이썬 커뮤니티의 지원 (강력한 표준 라이브러리)

- 광범위한 응용 분야 (웹 개발, 데이터 분석, 인공지능, 자동화 스크립트)

- 빠른 프로토타이핑

## 파이썬 인터프리터 사용법

- shell이라는 프로그램으로 한 번에 한 명령어씩 입력해서 실행

- 확장자가 .py인 파일에 작성된 파이썬 프로그램을 실행

```python
python 파일명
```

---

## 용어

- **표현식**: 코드 조각 3+5

- **값(피연산자)**: 표현식이 평가된 결과 8

- **평가**: 표현식을 순차적으로 실행하여 값을 얻는 과정

- **문장**: 실행 가능한 동작을 기술하는 코드 (조건문, 반복문, 함수 정의 등)

- **Data types**: 변수나 값이 가질 수 있는 데이터의 종류를 의미
  
  - 필요 이유
    
    - 값 구분, 어떻게 다뤄야 하는지
    
    - 각자에게 적합한 도구
    
    - 코드 읽는 사람이 변수 의도 쉽게 파악

- **산술연산자** (** -(음수) * / // % + -) <- 우선 순위

---

## 값이 저장되는 법

- **변수**: 값을 참조하기 위한 이름 (객체가 있는 방 주소의 별명)

- **변수명 규칙**
  
  - 영문 알파벳, 언더스코어(-). 숫자로 구성
  
  - 숫자로 시작할 수 없음
  
  - 대소문자를 구분
  
  - 파이썬 내부 예약어는 사용 불가능

- **할당**(덮어씌우면 **재할당**): 표현식을 통해 변수에 값을 저장

- **객체**: 타입을 갖는 메모리 주소 내 값
