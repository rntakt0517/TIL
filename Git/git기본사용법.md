---

## 위치 이동

```bash
cd 파일명/
```

> 상위 폴더는 `..`으로

---

## 생성

- 파일 생성

```bash
touch 문서명
```

> .txt = 문서 파일
> 
> .md = 마크다운 파일
> 
> .py = 파이썬 파일

- 폴더 생성

```bash
mkdir 파일명
```

---

## 삭제

- 파일 삭제

```bash
rm 문서명
```

- 폴더 삭제

```bash
rm -r 폴더명
```
