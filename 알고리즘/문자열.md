# 문자열

---

## 문자의 표현

- 코드체계 : 영어가 대소문자 합쳐서 52자 이므로 6(64가지)비트면 모두 표현 가능

- 출력 가능 아스키 문자 (32~126)

- 확장 아스키 : 표준 문자 이외의 악센트, 도형, 특수 문자, 특수 기호 등 부가적 128개

> 1/0 구분은 bit, 이게 8개 있으면 Byte인데 주소 단위임

- 유니코드 : 다국어 처리를 위한 표준
  
  - 유니코드도 다시 Character Set으로 분류됨
  
  - 적당한 외부 <mark>인코딩</mark>이 필요

- ord(A) = 65

- chr(65) = A
