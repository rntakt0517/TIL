## 복합 연산자

- 연산과 할당이 함께 이뤄짐

```python
  a += b
  -> a = a + b
```

## 비교 연산자

- `is` 같음, `is not` 같지 않음 : 주소까지 보는 강한 같음

---

## 논리 연산자

- `and, or, not`

- <mark>단축평가</mark>
  
  - <mark>`and` False 나오면 그만</mark>
  
  - <mark>`or` True 나오면 그만</mark>
  
  - <mark>이유: 코드 실행 최적화, 불필요한 연산 피하고자</mark>

---

## 멤버십 연산자

- `in, not in`

- 특정 값이 시퀀스나 다른 컬렉션에 속하는지 여부를 확인

---

## 시퀀스형 연산자

- 시퀀스 간 연산에서 다른 역할
  
  - `+` : 결합 연산자
  
  - `*` : 반복 연산자
